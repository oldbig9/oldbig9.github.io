## 二叉搜索

1. 递归
```go
func binarySearch(nums []int, target int) bool {
	if len(nums) == 0 {
		return false
	}
	mid := len(nums) / 2
	if mid == 0 && nums[mid] != target {
		return false
	}
	if nums[mid] > target {
		return binarySearch(nums[:mid], target)
	} else if nums[mid] < target {
		return binarySearch(nums[mid+1:], target)
	} else {
		return true
	}
}
```

2. 迭代

迭代法可以很方便的返回查找目标的下标
```go
func binarySearch2(nums []int, target int) int {
	i, j := 0, len(nums)-1
	for i < j {
		mid := i + (j-i)/2
		if nums[mid] == target {
			return mid
		} else if nums[mid] > target {
			j = mid - 1
		} else {
			i = mid + 1
		}
	}

	return -1
}
```

## 算法题-切木头

```go
func cutWoods(woods []int, k int) (max int) {
	sort.Ints(woods)
	i, j := 1, woods[len(woods)-1]
	for i <= j && i > max {
		mid := i + (j-i)/2
		count := 0
		for _, v := range woods {
			count += v / mid
		}
		if count >= k {
			max = mid
			i = mid + 1
		} else {
			j = mid - 1
		}
	}

	return max
}
```
