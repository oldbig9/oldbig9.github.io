## 选择排序

每次将未遍历区域的最小数据放在已排序区域末尾

```go
func selectionSort(nums []int) {
    length := len(nums)
    for i:=0; i<length; i++ {
        k := i
        for j:=i+1; j<length; j++ {
            if nums[k] < nums[j] {
                k = j
            }
        }
        nums[k], nums[i] = nums[i], nums[i]
    }
}
```

## 冒泡排序

```go
func bubbleSort(nums []int) {
    length := len(nums)
    for i:=length-1; i>= 0; i-- {
        for j := 0; j < i; j++ {
            if nums[j] > nums[j+1] {
                nums[j], nums[j+1] = nums[j+1], nums[j]
            }
        }
    }
}
```
