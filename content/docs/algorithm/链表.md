## 链表反转

1. 虚拟一个prev节点，cur节点为当前节点（初始化时为head节点），next为下一个节点
2. 记录next节点，next := cur.Next
3. 当前节点next指向前一个节点，cur.Next = prev
4. prev节点后移，prev = cur
5. cur节点后移，cur = next

```go
package main

func main() {
	s := []int{1, 2, 3, 4}
	h := &ListNode{}
	cur := h
	for _, v := range s {
		cur.Next = &ListNode{Val: v}
		cur = cur.Next
	}

	r := reverseList(h.Next)
	for r != nil {
		println(r.Val)
		r = r.Next
	}
}

type ListNode struct {
	Val  int
	Next *ListNode
}

func reverseList(head *ListNode) *ListNode {
	cur := head
	var prev *ListNode
	for cur != nil {
		next := cur.Next
		cur.Next = prev
		prev = cur
		cur = next
	}

	return prev
}
```


## 链表分组反转

给定一个链表，将链表每k个节点进行反转，如果最后剩余节点不足k个，则保持原有顺序不变。
例 1->2->3->4->5->6->7->8, k=3时
结果为 3->2->1->6->5->4->7->8
